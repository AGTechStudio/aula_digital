# Aula Digital

Aplicación Android desarrollada para la gestión integral de actividades escolares por parte de docentes orientada al colegio Nuestra Señora de Guadalupe N°1147 Santa Fe, Argentina.

## 🚀 Descripción

Aula Digital es una solución pensada para facilitar la organización diaria en el aula, permitiendo gestionar alumnos, asistencia, meriendas y planificaciones desde un único lugar.

El proyecto fue desarrollado con enfoque práctico, simulando una necesidad real del entorno educativo.

---

## ⚙️ Funcionalidades principales

* 📋 Gestión de alumnos (alta, edición y eliminación)
* 📅 Calendario interactivo
* 🍎 Registro de meriendas por día
* 📝 Planificaciones con archivos (PDF, imágenes)
* 📊 Asistencia y seguimiento
* ☁️ Integración con Firebase
* 💾 Persistencia local con Room

---

## 🛠️ Tecnologías utilizadas

* Java (Android)
* Android Studio
* Firebase Authentication & Firestore
* Room Database (persistencia local)
* API REST (feriados)

---

## 💡 Problema que resuelve

Muchos docentes gestionan información diaria en papel o herramientas dispersas.

Aula Digital centraliza esa gestión en una app simple, mejorando la organización y reduciendo errores.
